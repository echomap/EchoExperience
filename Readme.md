EchoExperience
Addon/mod for Elder Scrolls Online
Outputs Experience gains to chat

Slash commmands:
/echoexp
/echoexp debug
/echoexp tab #  -- to contorl what chat tab to output to
/echoexp window #  -- to contorl what chat window to output to
